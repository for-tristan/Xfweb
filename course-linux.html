About This Course

Master Linux from the ground up — from basic command-line operations to advanced customization with Arch Linux and Hyprland. This course is designed for users who want to break free from restrictive operating systems and take full control of their computing experience.

What You'll Learn

Linux File System Hierarchy
Command Line Mastery (Bash/Zsh)
Arch Linux Installation & Setup
Wayland Display Server
Hyprland Tiling WM Configuration
Dotfiles Management with GNU Stow
Custom Rice & Theming

Course Modules

01

Linux Fundamentals

History of Unix/Linux, distro landscape, installation basics, file system navigation

02

Command Line Power User

Bash scripting, pipes/redirects, grep/sed/awk, process management, permissions

03

Arch Linux Deep Dive

Installation from scratch, pacman configuration, AUR usage, system tuning

04

Wayland & Hyprland Setup

Display server concepts, Hyprland config, keybinds, animations, rules

05

Dotfiles & Rice

GNU Stow, version control for configs, theming GTK/QT, terminal beautification

06

Advanced Customization

Custom scripts, status bars, notification daemons, launcher setup, productivity tools

Prerequisites

A computer capable of running Linux (or access to a VM), willingness to use the terminal, and patience to learn through experimentation. No prior Linux experience required!
